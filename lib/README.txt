This folder requires the following JS libraries.
Run setup.sh or download_libs.py to populate this folder.

Required files:
- pdf-lib.min.js       (from https://unpkg.com/pdf-lib@1.17.1/dist/pdf-lib.min.js)
- pdf.min.js           (from https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js)
- pdf.worker.min.js    (from https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.worker.min.js)
